# ttang1024.github.io

Personal portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

Live site: https://ttang1024.github.io

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** — build tool
- **Tailwind CSS v4** — styling
- **Motion** — animations
- **Lucide React** — icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

Output is generated in the `dist/` directory.

## Deploy

```bash
npm run deploy
```

Builds the project and deploys to GitHub Pages via `gh-pages`.

This repository also includes a GitHub Actions workflow at `.github/workflows/deploy.yml`.
